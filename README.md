# 🌍 Plan with Love

A beautifully designed **Indian Travel Tour App** built with **React + Vite**. Explore handpicked destinations across India, read about each city, and manage your travel wishlist interactively.

## ✨ Features

- 🗺️ Browse **7 stunning Indian destinations** — Agra, Jaipur, Goa, Mumbai, Delhi, Varanasi & Kerala
- 📖 **Read More / Show Less** toggle for each city description
- ❌ **Remove destinations** you're not interested in
- 🔄 **Refresh** to bring all destinations back
- 💅 Fully responsive design — works on mobile, tablet & desktop

## 🛠️ Tech Stack

- **React** — Component-based UI
- **Vite** — Fast development build tool
- **CSS3** — Custom styling with CSS variables, Grid & Flexbox
- **Google Fonts** — Playfair Display & Inter

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/plain-with-love.git

# Navigate into the project
cd plain-with-love

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser to view the app.

## 📁 Project Structure

```
src/
├── components/
│   ├── Card.jsx       # Individual tour card component
│   └── Tours.jsx      # Grid of all tour cards
├── App.jsx            # Root component with state management
├── data.jsx           # City data (name, info, image, price)
├── index.css          # Global styles
└── main.jsx           # App entry point
```
